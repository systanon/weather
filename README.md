# weather

[**demo**](https://systanon.github.io/weather/)

## Project setup

```bash
yarn install
```

## Start Project

#### add the .env.local file to run the project

```bash
VUE_APP_HOST="https://openweathermap.org"

VUE_APP_STATIC_HOST="http://openweathermap.org"

VUE_APP_APP_ID="d8f0e2f2539a0dba3cc174753ce2ed6a"

VUE_APP_API_URL="https://api.openweathermap.org"

VUE_APP_API_VER="2.5"
```
